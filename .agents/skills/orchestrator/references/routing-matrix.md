# Routing Matrix

Use this matrix to decide which specialist roles to invoke.

## Start With Researcher

- Unknown codebase area
- Competing technical options
- Need to compare libraries, APIs, or patterns
- Missing factual context

## Start With PM

- Vague or high-level feature idea
- Missing scope boundaries
- Missing acceptance criteria
- Need a delivery-ready plan

## Start With Architect

- Cross-module or cross-layer design work
- Need for new abstractions or boundaries
- Interface, schema, or workflow design
- Large refactor or migration planning

## Start With Developer

- Clear implementation request
- Bug fix with known target behavior
- Refactor with established scope

## Add Security Engineer

- Authentication or authorization changes
- Secret handling
- External input, uploads, or parsing
- Model provider or tool invocation changes
- New dependency or external integration

## Add QA

- Behavior changed
- New tests are needed
- Regression risk exists
- Release or merge confidence is requested
